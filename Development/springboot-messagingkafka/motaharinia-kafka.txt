Kafka:
In Kafka, producer will publish messages to topic, which is a stream of messages of a particular type. 
Consumer will subscribe to one or more topics of brokers by pulling the data.


1.Got to the Apache Kafka downloads page(https://kafka.apache.org/downloads) and download the https://kafka.apache.org/download the Scala 2.12 kafka_2.12-0.10.2.1.tgz

2.Next unzip it to a particular location (example: "D:\Program Files\kafka_2.12-2.5.0")

3.We will now start Apache Kafka. 
This Kafka installation comes with an inbuilt zookeeper. 
Zookeeper is mainly used to track status of nodes present in Kafka cluster and also to keep track of Kafka topics, messages, etc.
Open a command prompt and start the Zookeeper:
D:\Program Files\kafka_2.12-2.5.0\bin\windows\zookeeper-server-start.bat D:\Program Files\kafka_2.12-2.5.0\config\zookeeper.properties

4.Open a new command prompt and start the Apache Kafka:
D:\Program Files\kafka_2.12-2.5.0\bin\windows\kafka-server-start.bat D:\Program Files\kafka_2.12-2.5.0\config\server.properties

5.Next start the Spring Boot Application by running it as a Java Application.

6.Also Start the consumer listening to the java_in_use_topic-:
D:\Program Files\kafka_2.12-2.5.0\bin\windows\kafka-console-consumer.bat --bootstrap-server localhost:9092 --topic java_in_use_topic --from-beginning

7.Finally hit the url as follows- http://localhost:8080//javainuse-kafka/producer?message=test

8.This will trigger the message to be sent to the java_in_use_topic. We can see in the consumer started the message is recieved.

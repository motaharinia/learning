1.IntelijIDEA: Help -> Edit Custom Vm Options -> add these two line:
-Dfile.encoding=UTF-8
-Dconsole.encoding=UTF-8

2.IntelijIDEA: File -> Settings -> Editor -> File Encodings form "System default" to UTF-8. May be it affected somehow.

3.IntelijIDEA: File -> Settings -> Editor -> General -> Code Completion -> check "show the documentation popup in 500 ms"

4.IntelijIDEA: File -> Settings -> Editor -> General -> Auto Import -> check "Optimize imports on the fly (for current project)"

5.IntelijIDEA: File -> Settings -> Editor -> Color Scheme -> Color Scheme Font -> Scheme: Default & set font to "Tahoma"

6.IntelijIDEA: Run -> Edit Configuration -> Spring Boot -> XXXApplication -> Environment -> VM Options: -Dspring.profiles.active=dev

7.IntelijIDEA HotKeys:
Ctrl+Alt+L        : Format Code
Ctrl+D            : Duplicate Code Line
Shift+Alt+Up/Down : Move Code Line
Ctrl+Alt+O        : Remove Unused Import Lines
